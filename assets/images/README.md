# Images Directory

This directory contains theme images and assets.

## Structure

- `icons/` - SVG icons and small graphics
- `placeholders/` - Placeholder images for development
- `backgrounds/` - Background images and textures
- `logos/` - Logo variations and brand assets

## Image Optimization

All images should be optimized for web delivery:
- Use WebP format when possible
- Provide fallback formats (JPEG, PNG)
- Include multiple sizes for responsive images
- Compress images to reduce file size

## Naming Convention

Use descriptive, lowercase names with hyphens:
- `project-thumbnail-400x300.jpg`
- `news-featured-600x400.webp`
- `award-certificate-placeholder.png`
